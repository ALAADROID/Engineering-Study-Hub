## Hypothesis Testing:

**When we see:**
* claim  
* test  
* is it less than / greater than
* $H$<sub>o</sub> , $H$<sub>1</sub>  
> THIS = Hypothesis test

<br>

### Steps (always same)

**1️⃣ Write hypotheses:**  
* H<sub>o</sub>: main claim
* H<sub>1</sub>: what we test  
> If que. says *H<sub>1</sub><15*, then main one becomes *H<sub>0</sub>=15* auto.
> | Claim | H₁        | H₀        |  
> | ----- | --------- | --------- |  
> | <     | μ < value | μ = value |  
> | >     | μ > value | μ = value |  
> | ≠     | μ ≠ value | μ = value |  


<br>

**2️⃣ Choose type:**  
* < → left-tailed  
* \> → right-tailed  
* != → two-tailed

<br>

**3️⃣ Compute Z:**  
* $z$ = ($\overline{X}$ - $u$)/($s$/√ $n$) = ($\overline{X}$ - $u$)/($SE$)  

<br>

**4️⃣ Compare with critical Z:**  
* Given in question (like 1.96, 2.33…)

<br>

**5️⃣ Decision:**
* inside → accept
* outside → reject
