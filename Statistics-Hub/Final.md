## Simple Linear Regression:

### EXAM SOLVING STEPS:  
Most questions follow this.

- Step 1:  
Make table:  
| X | Y | X² | XY |
|:---|:---|:---|:---|
| - | - | -- | -- |


- Step 2:  
Calculate:  
∑X  
∑Y  
∑X<sup>2</sup>  
∑XY  

- Step 3:  
Find:  
`b1`
  
> using slope formula.

- Step 4:  
  Find:

  using
  
  `b0=Y−b1X`

- Step 5:  
  Write regression line:

`Y=b0+b1X`

- Step 6:
  Substitute X value to predict Y.
